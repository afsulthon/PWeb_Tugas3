Portofolio

Tutorial HTML

HTML merupakan singkatan dari Hyper Text Markup Language. HTML digunakan untuk mendesain halaman web menggunakan bahasa markup. HTML adalah kombinasi dari bahasa Hypertext dan Markup. Hypertext mendefinisikan hubungan antara halaman web.

Tutorial CSS

Cascading Style Sheets, yang sering disebut sebagai CSS, adalah bahasa yang dirancang sederhana yang dimaksudkan untuk menyederhanakan proses pembuatan halaman web yang rapi. CSS memungkinkan Anda untuk menerapkan gaya ke halaman web.

Tutorial PHP

Istilah PHP adalah singkatan dari PHP: Hypertext Preprocessor. PHP adalah bahasa skrip sisi server yang dirancang khusus untuk pengembangan web. PHP dapat dengan mudah disematkan dalam file HTML.

Tutorial JavaScript

Javascript dikembangkan oleh Brendan Eich pada tahun 1995. Pada awalnya, ini disebut LiveScript tetapi kemudian dinamai JavaScript. Javascript adalah otot dari struktur
